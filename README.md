# AI Resume Screening System

An AI-powered Resume Screening System built using **Python** and **Streamlit**. This application analyzes resumes, compares them with a Job Description, calculates an ATS Score, identifies matching and missing skills, and provides improvement suggestions.

---

## Features

- Upload Resume (PDF)
- Extract Resume Text
- Candidate Details
- Resume Analysis
- Resume Sections
- Resume Completeness Score
- Recommended Job Roles
- Interview Questions
- Job Description Analysis
- ATS Score Calculation
- Resume Statistics
- Skill Match Percentage
- Resume Rating
- Resume Suggestions
- Skill Match Pie Chart
- Skill Match Bar Chart
- Matching Skills
- Missing Skills
- Skill Gap Analysis
- Download ATS Report

---
